- 메인화면
    - 목표
        - 스텝설정은 목표추가 폼에서 추가할수있도록 변경
        - 목표를 클릭하면 목표의 상세정보를 확인할수있는 화면으로 이동
            - 목표 상세화면은 사용자가 설정한 목표명,목표내용,기간, 스텝을 확인가능하며, 스텝만 추가 및 변경도 가능
        - 목표삭제 기능을 추가하며, 목표에 연결된 스텝들도 목표가 삭제되면 같이 삭제된다. 목표를 삭제할때 버튼을 클릭하면 사용자에게 해당 내용을 안내하는 메시지를 보여준후 삭제여부를 선택 후 삭제진행
- 기능
    - LLM
        - 사용하는 프롬프트를 전부 한국어로 변경하고, 절대적으로 한국어로만 답변하도록 프롬프트 강화
