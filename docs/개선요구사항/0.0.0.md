- UI
    - 공통
        - 글꼴 변경 및 크기 키우기
        - 불필요하게 분리가 많이되어있는 카드UI는 필요한것끼리는 통합
    - 헤더
        - 왼쪽끝에는 햄버거 모양의 메뉴버튼을 추가
            - 해당 버튼을 클릭하게될 경우 왼쪽 사이드에서 슬라이드 애니메이션과 함께 사이드 메뉴가 나옴(notion의 사이드 메뉴와 유사)
                - 해당 메뉴에는 사용자분석,활동 캘린더, 초기화 메뉴 존재
        - 헤더의 언더라인은 삭제
        - 현재 우측에 존재하는 캐릭터 버튼 삭제
    - 메인 컨텐츠
        - 캐릭터 정보카드 
            - UI를 사용자의 목표,레벨,현재 경험치를 지금보다 한눈에 확인 가능하도록 수정
        - 활동기록
            - 기록 저장시 LLM과 통신하여 분석중이라는걸 사용자가 알수있는 UI추가(예를 들어 스피너)
        - 캘린더
            - 년,월을 보다 자유롭게 선택가능하도록 변경
            - 셀을 정사각형으로 변경
            - 사용자가 추가한 목표의 표시 색상을 현재는 해당 날짜들에 점으로 표시하고있지만,
            해당 셀 자체를 사용자가 선택한 색상으로 변경(다만 활동이 추가된걸 표시 및 활동 개수를 표시하는걸 고려하여 색상 적용)
            - 전체적인 UI는 노션의 캘린더UI를 유사하게 변경
- 기능
    - 첫화면인 캐릭터 목록화면은 삭제하고, 생성된 캐릭터가 없을경우에는 이름을 입력하여 캐릭터를 생성하고, 있을경우에는 캐릭터정보창으로 이동
