- 캐릭터 기능
    - Phaser.js를 사용해서 도트캐릭터를 구현하여 적용
    - 현재 적용되는 캐릭터 단계를 좀더 세분화해서 1랩업마다도 캐릭터의 변화가 가능한지 확인하고 가능하다면 수정
    - 캐릭터 성장 테스트를 해볼수있는 페이지를 생성후 캐릭터성장테스트기능 구현
    - 도트캐릭터를 구현하는데 필요한 내용이 있다면 질문을통해 답을 얻어라.
- 불필요 기능 개선 및 삭제
    - 기존에 있던 2개이상의 캐릭터를 고려한 기능은삭제하고, 해당 기능으로 구현된 기능은 모두 싱글캐릭터로 변경(예를들어 메뉴를 클릭해서 캘린더를 갈경우 아직도 캐릭터를 선택하라고나옴)

- 목표기능 개선
    - 현재는 목표명,시작일,종료일,연결할 능력 이렇게만 존재하나 목표이름,목표내용,시작일,종료일로 수정
    - 목표내용에는 step설정이 가능하며, 스탭설정페이지는 별도로 추가한다. 
    - 각 스텝에는 날짜 설정 및 스텝별 내용을 추가할수있다.
    - 변경된 목표기능에 맞춰 기존에 존재하던 분석기능을 변경 

- 메뉴추가
    - 성장그래프
        - 일일 성장, 주간성장, 월간성장을 그래프로 확인할수있는 메뉴를 추가 

- cursor rule 수정
    - 모든 코드작성시에는 eslint를 맞추고, any타입 사용을 자제하도록 수정
