# Life RPG 서비스 핵심 기능 설계

## 개요

Life RPG는 **목표 관리 + 활동 기록 + AI 분석 + RPG 성장 시스템**을
결합한 생산성 서비스입니다.\
사용자의 활동을 분석하여 경험치(XP)와 성장 피드백을 제공하고, 캐릭터
성장 요소를 통해 지속적인 동기부여를 제공합니다.

------------------------------------------------------------------------

# 1. AI 코치 기능

사용자가 작성한 활동 기록을 AI가 분석하여 개선 방향을 제시합니다.

### 예시 입력

-   공부: 2시간
-   SNS 사용: 3시간
-   운동: 없음

### AI 분석 결과

-   집중도 낮음
-   SNS 사용 과다
-   운동 부족

### 추천

-   SNS 사용 시간 줄이기
-   25분 집중 공부 (Pomodoro)
-   가벼운 운동 추가

------------------------------------------------------------------------

# 2. 자동 XP 계산

단순 체크 방식이 아니라 **활동 내용과 시간 기반으로 XP를 계산**합니다.

### 예시

-   1시간 공부 → 40 XP
-   3시간 공부 → 120 XP
-   운동 30분 → 50 XP

AI 분석 결과에 따라 XP 보너스 또는 패널티를 적용할 수 있습니다.

------------------------------------------------------------------------

# 3. 스킬 시스템

활동 유형에 따라 특정 스킬 경험치가 증가합니다.

### 스킬 예시

-   IT
-   운동
-   독서
-   언어
-   인간관계

### 활동 예시

`TypeScript 공부`

→ IT Skill XP 증가

------------------------------------------------------------------------

# 4. 스탯 시스템

캐릭터의 능력치를 통해 성장 상태를 시각화합니다.

### 스탯 예시

-   INT → 학습 능력
-   STR → 운동 능력
-   CHA → 인간관계
-   WIS → 자기관리

------------------------------------------------------------------------

# 5. 레벨 시스템

XP가 일정 수준을 넘으면 캐릭터 레벨이 상승합니다.

### 레벨업 보상 예시

-   캐릭터 외형 변화
-   새로운 아이템
-   새로운 배경
-   새로운 기능 해금

------------------------------------------------------------------------

# 6. Streak 시스템

연속적으로 목표를 달성하면 보너스를 제공합니다.

### 예시

-   7일 연속 공부

보상

-   XP +100
-   특별 아이템

------------------------------------------------------------------------

# 7. AI 목표 생성

사용자가 목표를 입력하면 AI가 학습 계획을 자동으로 생성합니다.

### 예시

입력

`JLPT N2 공부`

AI 생성

-   Day 1: 단어 학습
-   Day 2: 문법 학습
-   Day 3: 독해 연습

------------------------------------------------------------------------

# 8. 성장 그래프

사용자의 성장 데이터를 시각적으로 보여줍니다.

### 제공 데이터

-   주간 XP
-   월간 XP
-   스킬 성장 그래프
-   활동 시간 분석

------------------------------------------------------------------------

# 9. 장비 시스템

아이템을 통해 캐릭터 성장에 영향을 줍니다.

### 아이템 예시

-   책
-   노트북
-   운동화
-   헬스장 멤버십

### 효과

-   공부 XP +10%
-   운동 XP +15%

------------------------------------------------------------------------

# 10. 챌린지 시스템

특정 기간 동안 목표 달성에 도전하는 이벤트입니다.

### 예시

30일 운동 챌린지

보상

-   XP
-   특별 아이템
-   캐릭터 스킨

------------------------------------------------------------------------

# 11. 커뮤니티 기능

사용자 간 협력 및 경쟁 요소를 제공합니다.

### 기능

-   파티
-   길드
-   공동 목표
-   친구 시스템

------------------------------------------------------------------------

# 12. AI 분석 리포트

사용자의 활동 데이터를 분석하여 정기 리포트를 제공합니다.

### 예시

최근 30일 분석

-   공부 부족
-   SNS 사용 과다
-   운동 부족

------------------------------------------------------------------------

# 13. AI 성향 분석

사용자의 행동 패턴을 분석하여 성향을 도출합니다.

### 예시

-   집중형
-   야행성
-   루틴형
-   탐구형

------------------------------------------------------------------------

# 14. AI 루틴 추천

사용자의 목표와 패턴을 기반으로 하루 루틴을 추천합니다.

### 예시

08:00 공부\
10:00 휴식\
10:20 공부

------------------------------------------------------------------------

# 15. AI 일일 미션

AI가 매일 새로운 미션을 제공합니다.

### 예시

오늘의 미션

-   30분 운동
-   단어 20개 학습
-   10분 명상

------------------------------------------------------------------------

# 결론

Life RPG 서비스는 다음 구조로 발전할 수 있습니다.

    목표 설정
    ↓
    활동 기록
    ↓
    AI 분석
    ↓
    XP 계산
    ↓
    캐릭터 성장
    ↓
    사용자 행동 분석
    ↓
    AI 코칭

이 구조는 **Habit Tracker + RPG + AI 코칭**을 결합한 형태로, 기존 생산성
서비스보다 높은 사용자 몰입도를 기대할 수 있습니다.
