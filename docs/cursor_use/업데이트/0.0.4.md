## 0.0.4

- 메인 화면 목표 UX 개선
  - 목표 추가 폼에서 바로 스텝을 함께 추가할 수 있도록 변경
  - 목표 리스트에서 목표 제목을 클릭하면 목표 상세/스텝 화면으로 이동
- 목표 상세 + 삭제 플로우 추가
  - 목표 상세 화면에서 목표명, 내용, 기간, 연결된 능력, 스텝 목록을 한 번에 확인
  - 목표 삭제 시 연결된 스텝도 함께 삭제되며, 삭제 전 한국어 안내 메시지 및 확인 다이얼로그 표시
- LLM 프롬프트 한글화 및 한국어 전용 응답 강화
  - 활동 분석/요약/XP 분석용 프롬프트를 전부 한국어로 정리
  - 시스템 메시지로 한국어 응답을 강제하는 패턴 정비
